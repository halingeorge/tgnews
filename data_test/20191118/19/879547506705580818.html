#PulsoEmpresarial Las bases de datos de clientes y el marketing de bases de datos

18 Nov 2019, 19:35 by Msc. Julio Cesar Vargas

Las organizaciones necesitan conocer a sus clientes, por lo que deben recopilar  información  sobre ellos y almacenarla para ejecutar el marketing de base de datos. Una base de datos de clientes es un conjunto organizado de información exhaustiva sobre los clientes individuales reales o potenciales, es accesible y manipulable para conseguir propósitos de marketing tales como la generación de clientes, su clasificación, la comercialización de productos o servicios o el mantenimiento de relaciones con el target. El marketing de base de datos de clientes es un proceso de confección, mantenimiento y utilización de bases de datos de clientes.

Algunas organizaciones confunden las listas de direcciones de los clientes con la base de datos. Una lista de direcciones de clientes no es más que un conjunto de nombres, direcciones y números telefónicos. Una base de datos de clientes incluye mucha más información.

Utilización de la base de datos

Para identificar clientes potenciales: muchas organizaciones generan ventas a través de la publicidad de sus productos y servicios. Las organizaciones selecciona, dentro de la base de datos, a los mejores clientes potenciales, se propone en contacto con ellos con el fin de convertirles en clientes reales.

Para decidir que clientes deberían recibir una oferta especial: las organizaciones están interesadas en identificar oportunidades de ventas, y realizar ventas cruzadas; para esto las empresas establecen criterios que describen a su cliente ideal para una oferta concreta, luego buscan en su base de datos de clientes a los que más se ajustan a esta oferta. Si registra el índice de respuestas de clientes, con el tiempo, la empresa podrá mejorar aún más su precisión al seleccionar al target group. Tras conseguir una venta, la organización puede establecer una secuencia automática de actividades como por ejemplo: enviar una nueva oferta, llamarlo para ofrecerle un descuento adicional, entre otras.

Para aumentar la lealtad de los clientes: las empresas pueden generar interés y entusiasmo recordando la preferencia de los clientes, enviando obsequios y cupones de descuento.

Para reactivar las compras de los clientes: las organizaciones pueden instalar programas de correo electrónico para enviar felicitaciones de cumpleaños a sus clientes, recordatorios de ciertas promociones y ofertas especiales de temporada, este base de datos de clientes puede ayudar a la empresa a hacer ofertas a tractivas y oportunas a sus clientes.
